observe package

public object SpanContext

Holds the spancontext map of opentracing implementations. Used as a parent span across services. SpanContext is just a representation of a propogated Span and can not be finished.

Field Name Data Type Default Value Description
contextMap map<string> [] The map containing information to build span contexts of opentracing implementations

public function extractSpanContext(map headers, string traceGroup) returns (SpanContext)

Creates a span context of a parent span propogated through request chaining

Parameter Name Data Type Description
headers map The map of headers
traceGroup string The kind of error. e.g. DBError
Return Type Description
SpanContext The SpanContext that was propogated from another service

public function extractSpanContextFromHttpHeader(Request req, string traceGroup) returns (SpanContext)

Method to save the parent span and extract the span Id

Parameter Name Data Type Description
req Request The http request that contains the header maps
traceGroup string The group to which this span belongs to
Return Type Description
SpanContext The SpanContext that was propogated from another service

public function startSpan(string serviceName, string spanName, map | null tags, REFERENCE_TYPE reference, Span | SpanContext | null parentSpan) returns (Span)

Starts a span and sets the specified reference to parent span

Parameter Name Data Type Description
serviceName string The service name of the process
spanName string The name of the span
tags map | null The map of tags to be associated to the span
reference REFERENCE_TYPE childOf, followsFrom, root
parentSpan Span | SpanContext | null Can be of type Span or SpanContext. If no parent null can be given
Return Type Description
Span The newly started span
